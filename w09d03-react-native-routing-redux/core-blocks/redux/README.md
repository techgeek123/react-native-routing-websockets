## Release 0

Create a counter application using Redux which have the following operations

- Increment the count on pressing **Increment** button
- Decrement the count on pressing **Decrement** button

## Release 1

Create **actions** in `actions/actions.js` and add all type of actions being performed

Create **reducers** in `reducers/reducers.js` which will check for the dispatched action type and execute and update the necessary action type.

Create **components** which are required for this application in `components` folder

Create a `store` and pass to all available components using `<Provider/>` without individually passing it to each and every component
