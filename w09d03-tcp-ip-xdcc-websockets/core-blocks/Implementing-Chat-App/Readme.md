# Chat Application with Web Sockets

## Information
In this challenge you'll learn why do we learn Web Sockets. What power does Web Socket give you compared to Socket.IO

Have a look at this blog post: [Node.js & WebSocket — Simple chat tutorial](https://medium.com/@martin.sikora/node-js-websocket-simple-chat-tutorial-2def3a841b61)

We will build this same application in this challenge.

## Release 0: Building the front end
- Every new user gets to pick their name. They are assigned a random color
- It will also post a log message that a new user with "name" just connected to our server

## Release 1: Node
- Implement a Node.js server
- Use some plugin to create a wrapper around WebSocket Protocol

## Release 2: WebSocket
